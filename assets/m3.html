Operações com Matrizes M×N

Soma: O(M·N) Mult: O(M·N·P) Álgebra Linear + Arrays 2D
Enunciado: Implemente operações de soma e multiplicação de matrizes com análise de complexidade. Para a soma: A[M×N] + B[M×N] = C[M×N]. Para a multiplicação: A[M×N] × B[N×P] = C[M×P].
Implementação
def soma_matrizes(A, B):
    M, N = len(A), len(A[0])
    return [[A[i][j] + B[i][j] for j in range(N)] for i in range(M)]  # O(M·N)

def mult_matrizes(A, B):
    M = len(A); N = len(A[0]); P = len(B[0])
    C = [[0]*P for _ in range(M)]
    for i in range(M):       # O(M)
        for j in range(P):   # O(P)
            for k in range(N): # O(N)
                C[i][j] += A[i][k] * B[k][j]
    return C  # Complexidade total: O(M·N·P)

Torre de Hanói — Recursivo vs Iterativo

Tempo: O(2ⁿ) Espaço Rec: O(n) Recursão + Pilha explícita
Enunciado: Mova n discos do pino A para o pino C usando o pino B como auxiliar. Nunca coloque um disco maior sobre um menor. Implemente e compare a versão recursiva com a iterativa (usando pilha explícita).
Intuição
A recorrência é T(n) = 2T(n-1) + 1. Resolvendo: T(n) = 2ⁿ - 1. Para n=10: 1.023 movimentos. Para n=64: mais de 18 quintilhões — levaria bilhões de anos. A versão iterativa usa uma pilha explícita para simular o call stack da recursão, com o mesmo número de movimentos mas sem overhead de chamadas de função.
Implementação
def hanoi_rec(n, origem, destino, auxiliar, movimentos=[]):
    if n == 1:
        movimentos.append((origem, destino))
        return
    hanoi_rec(n-1, origem, auxiliar, destino, movimentos)   # Move n-1 para aux
    movimentos.append((origem, destino))                        # Move o maior
    hanoi_rec(n-1, auxiliar, destino, origem, movimentos)   # Move n-1 de aux → dest
    return movimentos

# T(n) = 2T(n-1) + 1  →  T(n) = 2ⁿ - 1 movimentos
# Espaço: O(n) na call stack (profundidade da recursão)
def hanoi_iter(n, origem, destino, auxiliar):
    # Pilha explícita simula o call stack da recursão
    pilha = [(n, origem, destino, auxiliar)]
    movimentos = []

    while pilha:
        n, o, d, a = pilha.pop()
        if n == 1:
            movimentos.append((o, d))
            continue
        # Empilha em ordem inversa (pilha é LIFO)
        pilha.append((n-1, a, d, o))    # 3: move n-1 aux→dest
        pilha.append((1, o, d, a))       # 2: move disco maior
        pilha.append((n-1, o, a, d))    # 1: move n-1 orig→aux

    return movimentos  # Mesmos 2ⁿ-1 movimentos, sem recursão!
// C++ — versão recursiva clássica
#include <iostream>
using namespace std;

void hanoi(int n, char origem, char destino, char aux) {
    if (n == 1) {
        cout << "Mover disco 1 de " << origem << " para " << destino << endl;
        return;
    }
    hanoi(n - 1, origem, aux, destino);
    cout << "Mover disco " << n << " de " << origem << " para " << destino << endl;
    hanoi(n - 1, aux, destino, origem);
}

int main() {
    hanoi(4, 'A', 'C', 'B');
    return 0;
}
