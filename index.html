1. O Problema da Celebridade

Lógica O(n): Utilizamos uma abordagem de eliminação. Assumimos que a primeira pessoa é a candidata. Iteramos pela multidão: se o candidato atual conhece a próxima pessoa, ele não pode ser a celebridade. Ao final de uma passagem, temos um único candidato que é validado em uma segunda passagem. 
def encontrar_celebridade(matriz_conhecimento, n):
    candidato = 0
    
    # Fase 1: Eliminação em O(n)
    for pessoa in range(1, n):
        if matriz_conhecimento[candidato][pessoa] == True:
            candidato = pessoa

    # Fase 2: Validação em O(n)
    for pessoa in range(n):
        if pessoa != candidato:
            if matriz_conhecimento[candidato][pessoa] or not matriz_conhecimento[pessoa][candidato]:
                return -1
                
    return candidato

2. Eleição por Maioria Absoluta

Lógica O(n): Utilizando o Algoritmo de Votação de Boyer-Moore. Mantemos um contador: se o voto for igual ao do candidato atual, incrementamos; se diferente, decrementamos. Se zerar, trocamos o candidato. 
def vencedor_maioria_absoluta(votos):
    candidato_atual = None
    saldo_votos = 0

    for voto in votos:
        if saldo_votos == 0:
            candidato_atual = voto
            saldo_votos = 1
        elif voto == candidato_atual:
            saldo_votos += 1
        else:
            saldo_votos -= 1

    if sum(1 for v in votos if v == candidato_atual) > len(votos) // 2:
        return candidato_atual
    return None

3. Busca na Agenda Telefônica

Lógica O(n): Uma busca linear simples iterando sobre uma lista não ordenada de contatos. 
def buscar_amigo_agenda(agenda_lista, nome_buscado):
    for contato in agenda_lista:
        if contato['nome'] == nome_buscado:
            return contato['telefone']
    return "Contato não encontrado"

4. Troca de Mensagem Segura e Anônima

Lógica O(n): Criptografia Assimétrica aplicada de forma linear. Confidencialidade (Chave Pública do receptor) e Autenticidade (Chave Privada do remetente). 
def preparar_mensagem_segura(mensagem, chave_priv_remetente, chave_pub_receptor):
    mensagem_processada = []
    
    for caractere in mensagem:
        char_assinado = assinar_caractere(caractere, chave_priv_remetente)
        char_seguro = criptografar_caractere(char_assinado, chave_pub_receptor)
        mensagem_processada.append(char_seguro)
        
    return mensagem_processada

5. Otimização da Distribuição de Jornais

Lógica O(n): Iteramos sobre as n bancas calculando a distância para os 3 pontos fixos. A complexidade O(3 * n) simplifica-se para O(n). 
import math

def otimizar_rotas_jornais(bancas, pontos_distribuicao):
    alocacao = {ponto: [] for ponto in pontos_distribuicao}
    
    for banca in bancas:
        menor_distancia = float('inf')
        melhor_ponto = None
        
        for ponto in pontos_distribuicao:
            dist = math.sqrt((banca[0]-ponto[0])**2 + (banca[1]-ponto[1])**2)
            if dist < menor_distancia:
                menor_distancia = dist
                melhor_ponto = ponto
                
        alocacao[melhor_ponto].append(banca)
        
    return alocacao

Motivações e Legado do Projeto

A escolha de desenvolver este site como ferramenta de apresentação, no lugar de slides tradicionais, baseia-se em dois pilares fundamentais para o ensino e aprendizado contínuo:

  • Atemporalidade e Compartilhamento: Apresentações de slides frequentemente se perdem nos arquivos locais após o fim do semestre. Ao hospedar estas soluções na web, criamos um repositório vivo e acessível. Nosso objetivo é que estudantes de semestres futuros possam consultar este material de forma clara, garantindo que o conhecimento desenvolvido na disciplina continue ajudando outras pessoas a compreenderem a análise de complexidade.

Sua sintaxe enxuta, legível e livre de excessos de formatação (como chaves e pontos-e-vírgulas) assemelha-se profundamente ao **pseudo-código** cobrado e utilizado nas provas teóricas da disciplina. Dessa forma, os algoritmos aqui expostos servem não apenas para compilar em uma máquina, mas como um molde mental facilmente transcritível para o papel em momentos de avaliação.
